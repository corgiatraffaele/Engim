Utilizzando i metodi del dom creare un form con 3 input di tipo testo ed un bottone per il submit.
i 3 input saranno:
nome, password e ripeti password
al click controllare tramite javascript che i tre input non siano vuoti,
che il contenuto di password e ripeti password sia almeno di 8 caratteri e che siano =
Se tutti i controlli passano eseguire il submit del form se alcuni controlli non passano mostrare un messaggio di errore.
